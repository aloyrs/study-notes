# Truth set

# ∃!

∃! = 'there exists a unique' / 'there is one and only one'

# ⇒

double arrow to indicate
implicit quantification symbolically.

For instance, x > 2 ⇒ x^2 > 4 ,
expresses “If x > 2 then x^2 > 4”
